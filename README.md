Ejercicios adicionales (Pagina 152 de "Introducción a la programación")

1. [intercambio.psc] Leer dos números A y B e intercambiar el valor de sus variables.
2. [calcularDiasHorasMinutos.psc] Dado un tiempo en minutos, calcula los días, horas y minutos que le corresponden.
3. [negativoYMaximo.psc] Leer 5 números enteros y decir si alguno ha sido negativo e indicar el valor máximo introducido.
4. [digitosContador.psc] Dado un número, determinar cuántos dígitos tiene.
5. [numerosMinorYMayor.psc] Leer una cantidad ‘N’ mayor que cero, y luego introducir ‘N’ números enteros positivos. Se pide imprimir el mayor y el menor y las veces que aparece cada uno.
